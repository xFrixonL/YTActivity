# YTActivity

A high-performance YouTube activity analysis tool linked to **YTDataScraper**. This module automates the process of qualifying leads by scanning channel upload frequencies within the last 30 days.

## 📌 Overview

This tool is designed as an extension for **YTDataScraper**. While the main scraper gathers channel metadata, **YTActivity** deep-dives into the "Videos" tab to determine if a creator is currently active, allowing for high-efficiency lead qualification.

## 🚀 Features

* **Intelligent Scrolling:** Automatically scrolls only if the first load contains exclusively recent videos.
* **Smart Stop:** Halts scanning immediately upon reaching videos older than 30 days.
* **Lead Qualification:** Automatically categorizes channels into activity levels (HIGH, MEDIUM, LOW, NULA).
* **Excel Integration:** Seamlessly reads from and writes to `.xlsx` files.
* **Real-time Tracking:** Console output with timestamps and progress counters `[current/total]`.

## 🛠️ Prerequisites

This script requires an input file generated by **YTDataScraper** (typically named `ytexcel.xlsx`). The input file must contain at least the following columns:

* `url`: The full URL of the YouTube channel.
* `name`: The name of the channel.
* `observaciones`: Used to skip "No encontrada" (404) channels.

## 📦 Installation

### 1. Clone the repository

`git clone https://github.com/xFrixonL/YTActivity.git`
`cd YTActivity`

### 2. Set up a Virtual Environment

**Windows:** `python -m venv venv`
`.\venv\Scripts\activate`

**Mac/Linux:** `python -m venv venv`
`source venv/bin/activate`

### 3. Install Dependencies

`pip install -r requirements.txt`

### 4. Install Playwright Browsers

`playwright install chromium`

## 📂 Usage

1. Place your `ytexcel.xlsx` (generated by YTDataScraper) in the project folder.
2. Run the script:`python main.py`
3. The script will generate a `resultado_final.xlsx` with the updated activity metrics.

## 📊 Activity Logic

The script classifies channels based on the number of videos uploaded in the **last 30 days**:

| Videos (30d)    | Level | Description                 |
| :-------------- | :---- | :-------------------------- |
| **7+**    | ALTA  | Highly active creator.      |
| **3 - 6** | MEDIA | Consistent uploader.        |
| **1 - 2** | BAJA  | Low frequency.              |
| **0**     | NULA  | Inactive in the last month. |
